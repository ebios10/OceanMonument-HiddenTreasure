海底神殿の秘宝 — Step 1: ゲームエンジン基盤

🏆 ゲーム終了

📖 行動日誌
遊び方 — 海底神殿の秘宝

ゲームの目的

突如海上に浮上した海底神殿から秘宝を持ち帰ることが目的です。
秘宝（銀の鍵または輝くトラペゾヘドロン）を獲得した瞬間にゲームが終了し、
最終的に勝利点が最も多いプレイヤーが勝利します。

プレイヤー人数

2〜6人でプレイします。AIはノーマル・ハード・ストロングの3段階。

準備

各プレイヤーはスターターデッキ（10枚）を持ちます：

  • 初級探索者 ×4（攻撃+1 / 純収益-1）
  • 古びた旧神の印 ×2（守護+1 / 重さ1 / 純収益-1）
  • ナイフ ×2（攻撃+1 / 重さ1 / 純収益-1）
  • 初級防具 ×2（防御+1 / 重さ1 / 純収益-1）

開始時に山札から8枚を手札として引きます。手番順に関わらず常に8枚です。

手番の流れ

スタートプレイヤーから時計回りに手番を行います。全員が1回ずつ手番を終えると次の週へ進みます。手番では以下から1つを選んで実行します：

  • 海底神殿に入る — 神話生物等と戦い宝を手に入れる
  • 港を訪れる — カードを購入してデッキを強化する
  • 大学を訪れる — 鑑定・寄贈して勝利点を確定しデッキを整える
  • 休息する — カードを破棄・訓練してデッキを整える

ゲーム終了と勝利

いずれかのプレイヤーが秘宝（銀の鍵または輝くトラペゾヘドロン）を獲得した瞬間にゲームが終了します。
その後、全プレイヤーの勝利点を集計し、最多のプレイヤーが勝利します。
※輝くトラペゾヘドロンは「狂気カード1枚につき+1点」の特殊効果があります（獲得者の手札の狂気を集計）。

同点時のタイブレーク（順に適用）

  1. 秘宝を持っているプレイヤーが勝利
  2. 秘宝なしで同点の場合、宝剣・宝冠・宝盃を多く持つプレイヤーが勝利
  3. 上記も同数の場合、下位に秘宝を持つプレイヤーがいればその秘宝獲得者が勝利

週イベント

各週の開始時に週イベントが1枚公開されます。イベントの効果はその週の全プレイヤーに適用されます。
イベントカードをシャッフルして使用。尽きたら再シャッフルします。

  • 👹 神話生物の活性化（神殿系）：神殿探索時に狂気カード1枚が捨て札に追加される
  • ✨ 探索者の奮起（神殿系）：神殿で勝利した時、手札の狂気カードを1枚除去できる
  • 🔗 連続探索（神殿系）：神殿フェーズで勝利後、続けてもう1度挑戦できる
  • ⚓ 凪の港（港系）：購入コスト-1（下限1）
  • 🚢 補給船到着（港系）：購入回数+1
  • ⛈ 嵐の海峡（港系）：購入コスト+1
  • 🌙 安息の夜（休息系）：休息で破棄できるカードが最大3枚に
  • 💧 回復の泉（休息系）：休息で手札の狂気カードを全て除外できる（無料）
  • 📜 古の知識（休息系）：訓練コストが💵5になる
  • ⚡ 特訓の時（休息系）：訓練で戦闘力+2（コスト💵6）
  • 🔮 霊的な加護（全体系）：守護を持つ道具の守護値+1
  • ☀ 快晴の朝（全体系）：神殿の1回目の探索を偵察済みにする
  • 🌪 激動の日（全体系）：神殿の1回目を偵察済みにする・守護+1が同時発生
  • ☀ 平穏の日（全体系）：特殊効果なし

デッキについて

全カードは常に「場札」として公開状態で保持されます。手番終了時のドロー・捨て山サイクルはありません。

正気度システム

設定でオンにすると正気度（SAN値）が有効になります。

正気度の計算方式は設定の「正気度の計算方式」で選択できます。

  • 通常モード（デフォルト）：100 − （デッキ汚染率×1.5）
  • 狂気モード：100 − （デッキ汚染率×3.0）
  • 神話生物等との戦闘で敗北するたびに正気度チェックが発生します。
  • 1〜100のダイスを振り、出目が正気度を超えた場合は「狂気の発作」が起こり追加の狂気カードが捨て山に加わります。
  • 追加枚数は出目に依存：100→6枚（特別演出） ／ 99〜95→5枚 ／ 94〜80→4枚 ／ 79〜50→3枚 ／ 49以下→2枚
  • 正気度チェックで正気度が0になると「永久的狂気」となります。人間プレイヤーはゲームオーバー、AIは手番から脱落します。
  • 永久的狂気になったプレイヤーの勝利点は0となり、狂気カード1枚につき−1点が加算されます。
  • 正気度と汚染率はゲーム画面と戦闘準備画面に常時表示されます。

おすすめモード

設定でオンにするとAIが推奨する行動ボタンに「おすすめ」の吹き出しが表示されます。

  • 推奨基準：戦闘力20以上（神話生物デッキ）または戦闘力33以上（全デッキ）で勝てる場合に神殿をおすすめ。
  • 金貨1以下の場合は港をおすすめしません。
  • 警告が出る選択肢（戦闘力5以下または金貨0）はおすすめになりません。

海底神殿に入る

  1. 手札を全て公開します。各探索者への装備は自動で割り当てられます。
    各探索者に武器1・防具1・道具1が装備されます（アイテムの重さ ≤ 探索者の筋力のものに限る）。
    火器（拳銃・自動小銃）は「火器使用可能」の探索者のみ装備されます。
  2. 神殿に積まれたカードをオープンします：
    • 宝物（宝冠・宝盃・宝剣）→ 即座に獲得、手札へ。
    • 秘宝（銀の鍵・輝くトラペゾヘドロン）→ 即座に獲得、ゲーム終了。
    • 神話生物・邪神の化身→ 遭遇演出後、戦闘へ。
    • 探索イベント（遺物発見）→ 調査か無視かを選択。
  3. 戦闘を解決します：

    ①遭遇時に狂気カードが追加される（戦闘前）
      通常の神話生物 → 2枚（手札1＋捨て山1）
      邪神の化身 → 3枚（手札2＋捨て山1）、さらに特殊行動を実行

    ②合計戦闘力を計算（全探索者の戦闘力 + 装備の戦闘力）

    ③狂気ペナルティ：手札の狂気カード枚数に応じた 2の累乗 のペナルティ
      （狂気1枚→2 / 狂気2枚→4 / 狂気3枚→8 / 狂気4枚→16…）
      旧神の印などの守護1毎に狂気1枚分を相殺（ペナルティを軽減）

    ④戦闘力 = 合計戦闘力 − 狂気ペナルティ

    ⑤戦闘力 ≥ 神話生物強さ → 勝利：神話生物カードを捨て山へ

    ⑥戦闘力 ＜ 神話生物強さ → 敗北
      神話生物デッキのカード：デッキの一番下に戻り次のカードが現れる
      秘宝デッキのカード：倒すまで居座り続ける
      敗北時は正気度チェックが発生（正気度システム有効時）

  4. 邪神の化身と戦う場合：
    • 装備している宝物カード（宝冠・宝盃・宝剣）の相手が邪神の化身の場合、ボーナスが戦闘力に加算されます。
    • 遭遇時に狂気カード3枚が追加された後、邪神の化身がランダムで特殊行動を1つ実行します：
      武器破壊／防具破壊／狂気発動（+1枚）／静観／自己強化１（強さ+4）／自己強化２（強さ+8）／連続行動（低確率）／３連続行動（低確率）
      連続行動：5種の行動（武器破壊・防具破壊・狂気発動・自己強化×2種）からランダムに2つを続けて実行します。
      ３連続行動：同5種の行動からランダムに3つを続けて実行します。
      行動後に戦闘力を再計算し、その結果で勝敗を判定します。
  5. 手番終了。場札はそのまま次のターンへ持ち越されます。

神殿の2種類のデッキ

神話生物デッキ：通常神話生物・宝物・探索イベント・邪神の化身1体が混在。倒せなかった場合はデッキ底に戻り、次のカードが現れます。

秘宝デッキ：残りの邪神の化身・秘宝が入っています。倒せなかった邪神の化身は居座り続け、倒すまで秘宝にたどり着けません。

探索イベント（遺物発見）

神話生物デッキから探索イベントカードが出ると、2択を迫られます：

  • A：無視する — 遺物は獲得できないが、狂気カードも増えない（安全）
  • B：獲得する — 遺物を獲得できる（考古学者・魔術師が助力）
      同行者なし：未鑑定品を手札に獲得 ＋ 狂気×1
      考古学者同行：未鑑定 or 偽物 or コモンを取得 ＋ 狂気なしで獲得
      魔術師同行：鑑定済遺物を取得（金貨値・VP+1）＋ 狂気×1（偽物の場合は狂気なし・破棄）

遺物カードはレアリティがあり、未鑑定品は大学で鑑定するまで内容不明。

港を訪れる

  1. 手札を全て公開します。
  2. 公開したカードの港効果が自動で発動し、金貨値を合計します。
    例）商人は「購入コスト-2・追加1枚購入」、補給船到着イベントは「追加1枚購入」が自動適用されます。
  3. 港では毎回ランダムで3名の仲間候補と3種の品揃え（武器・防具・その他各1種）が表示されます。
    合計金貨値以下のコストのカードを購入できます。通常は1枚（商人・補給船到着等で増加）。
    ※レア・高性能なカードほど品揃えに出にくい仕様です。
    購入したカードは手札（場札）へ加わります。未使用金貨は消滅します。
  4. 手番終了。場札はそのまま次のターンへ持ち越されます。

大学を訪れる

探索で得た遺物や討伐した神話生物を鑑定・寄贈できる場所です。
手番の選択肢の一つで、選択後は「大学から帰る」ボタンで手番が終了します。

  1. ゲーム画面から「🎓 大学を訪れる」を選択します。
    遺物・神話生物がデッキにない場合は訪問できません。
  2. デッキ・手札・捨て山に含まれる遺物および神話生物が一覧で表示されます。
  3. 鑑定・寄贈を行い、「大学から帰る →」で手番終了。
    場札はそのまま次のターンへ持ち越されます。

遺物の鑑定

未鑑定品を💵5で即時鑑定できます。
考古学者が手札にいる場合、鑑定料が💵4（-1割引）になります。

  • 鑑定結果：偽物 / 本物
  • 本物の場合：コモン ／ アンコモン ／ レア ／ エピック
  • 偽物が出た場合 → 「大学に引き取らせる」で処分できます（VP 0）
  • 考古学者が手札にいる場合 → 偽物引き取りで+1VP獲得

遺物の寄贈

鑑定済みの遺物を大学に寄贈すると勝利点が確定し、デッキから永久に取り除かれます（デッキ圧縮）。

遺物のレアリティと勝利点

  • コモン：3VP
  • アンコモン：5VP
  • レア：7VP
  • エピック：9VP

遺物は偶像（🗿）・書物（📚）・薬品（⚗️）の3種類があります。

神話生物の寄贈

討伐した神話生物（デッキ・手札・捨て山に残っているもの）を大学に寄贈できます。
勝利点が確定しデッキから取り除かれるため、デッキ圧縮と勝利点確定の両方の効果があります。

神話生物の勝利点は種類によって異なります（例：グール2VP、ショゴス4VP、邪神の化身7VP）。

寄贈による勝利点の増加

鑑定済みの遺物、討伐した神話生物を大学に寄贈することで、本来の勝利点に＋１点を加えます。

例：グール2VP+1VP=3VP、ショゴス4VP+1VP=5VP、書物（レア）7VP+1VP=8VP

協賛を募る

大学には協賛プールという資金プールがあります。大学を訪れるたびに基本値+3が加算され、カードを寄贈するたびにそのカードの純収益の絶対値が追加されます。

  • 「🤝 協賛を募る」ボタン：1ターンに1回のみ実行可。プール全額を即時獲得してリセット（💰 = プール残額）。
  • 後援額：毎ターン自動加算される安定収入。初期値+3。大学で寄贈するたびに+1増加し、協賛とは別扱い。
  • 持ち越し：協賛を使わなかった場合、ターン終了時にプールが+20%増加して次週に持ち越される。

例）ショゴス（純収益-12）を寄贈 → プール+12。翌週使わなければさらに+20%増。

休息する

休息では今週の行動として以下の3種別から1つを選んで実行します。選択後は「休息を終える →」で手番終了。

🚪 解雇（最大2人）

手札の探索者を最大2人選んで解雇できます。解雇した探索者はデッキから永久に除外されます。
維持費（純収益の支出）が減り、毎ターンの収入が改善されます。

戦力が落ちる反面、デッキの赤字を解消したいときに有効です。

🗑 破棄（最大2枚）

手札の武器・防具・道具カードを最大2枚選んで破棄できます。除外されてデッキが軽くなります。
※「安息の夜」イベント中は最大3枚まで破棄できます。

  • 解体業者が同行している場合：最大3枚・狂気カードと通常カードの混在破棄が可能。

💊 治療（最大2枚）

手札の狂気カードを最大2枚選んで除外できます。デッキの汚染率が下がり正気度が回復します。
※「回復の泉」イベント中は手札の狂気カードを全て無料で除外できます。

  • 精神科医が同行している場合：治療モードで手札の狂気カードを全て除外できます（無制限）。

訓練（行動とは別に随時実行可）

探索者カードに表示された訓練ボタンから実行。💵6消費するごとに攻撃力+1。
初回のみ、攻撃力+1に加えて筋力も+1されます。探索者1人につき1回のみ。

  • 傭兵が同行している場合：初回訓練無料・各探索者の2回目訓練で筋力+1追加。
  • 古の知識イベント中：訓練コストが💵5に減少。
  • 特訓の時イベント中：訓練で攻撃力+2（コスト💵6）。
