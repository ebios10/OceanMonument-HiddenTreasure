海底神殿の秘宝 — Step 1: ゲームエンジン基盤

🏆 ゲーム終了

📖 行動日誌
遊び方 — 海底神殿の秘宝

ゲームの目的

突如海上に浮上した海底神殿から秘宝を持ち帰ることが目的です。
秘宝（銀の鍵または輝くトラペゾヘドロン）を獲得した瞬間にゲームが終了し、
最終的に勝利点が最も多いプレイヤーが勝利します。

プレイヤー人数

2〜6人でプレイします。AIはノーマル・ハード・ストロングの3段階。

準備

各プレイヤーはスターターデッキ（12枚）を持ちます：

  • 初級探索者 ×6（攻撃1）
  • 古びた旧神の印 ×2（守護1・金貨値2）
  • ナイフ ×2（攻撃+1・金貨値1）
  • 初級防具 ×2（防御+1・金貨値2）

開始時に山札から8枚を手札として引きます。手番順に関わらず常に8枚です。

手番の流れ

スタートプレイヤーから時計回りに手番を行います。全員が1回ずつ手番を終えると次の週へ進みます。手番では以下から1つを選んで実行します：

  • 海底神殿に入る — 神話生物等と戦い宝を手に入れる
  • 港を訪れる — カードを購入してデッキを強化する
  • 休息する — カードを破棄・訓練してデッキを整える

ゲーム終了と勝利

いずれかのプレイヤーが秘宝（銀の鍵または輝くトラペゾヘドロン）を獲得した瞬間にゲームが終了します。
その後、全プレイヤーの勝利点を集計し、最多のプレイヤーが勝利します。
※輝くトラペゾヘドロンは「狂気カード1枚につき+1点」の特殊効果があります（獲得者の手札の狂気を集計）。

同点時のタイブレーク（順に適用）

  1. 秘宝を持っているプレイヤーが勝利
  2. 秘宝なしで同点の場合、宝剣・宝冠・宝盃を多く持つプレイヤーが勝利
  3. 上記も同数の場合、下位に秘宝を持つプレイヤーがいればその秘宝獲得者が勝利

週イベント

各週の開始時に週イベントが1枚公開されます。イベントの効果はその週の全プレイヤーに適用されます。
イベントカードをシャッフルして使用。尽きたら再シャッフルします。

  • 👹 神話生物の活性化（神殿系）：神殿探索時に狂気カード1枚が捨て札に追加される
  • ✨ 探索者の奮起（神殿系）：神殿で勝利した時、手札の狂気カードを1枚除去できる
  • 🔗 連続探索（神殿系）：神殿フェーズで勝利後、続けてもう1度挑戦できる
  • ⚓ 凪の港（港系）：購入コスト-1（下限1）
  • 🚢 補給船到着（港系）：購入回数+1
  • ⛈ 嵐の海峡（港系）：購入コスト+1
  • 🌙 安息の夜（休息系）：休息で破棄できるカードが最大3枚に
  • 💧 回復の泉（休息系）：休息で手札の狂気カードを全て除外できる（無料）
  • 📜 古の知識（休息系）：訓練コストが💵5になる
  • ⚡ 特訓の時（休息系）：訓練で戦闘力+2（コスト💵6）
  • 🔮 霊的な加護（全体系）：守護を持つ道具の守護値+1
  • 🌀 快晴の朝（全体系）：神殿探索時は追加ドロー+2枚（手番開始時の引き直しなし）
  • 🌪 激動の日（全体系）：神殿で追加ドロー2枚・守護+1・連続探索可能が同時に発生する
  • ☀ 平穏の日（全体系）：特殊効果なし

デッキについて

山札が足りなくなったとき、捨て山をシャッフルして新しい山札にします。手番終了時は全手札を捨て山へ置き、新たに8枚ドローします。

正気度システム

設定でオンにすると正気度（SAN値）が有効になります。

正気度は100 − （デッキ汚染率×2） − （手札の狂気カード枚数×10）で算出します。

  • 神話生物等との戦闘で敗北するたびに正気度チェックが発生します。
  • 1〜100のダイスを振り、出目が正気度を超えた場合は「狂気の発作」が起こり追加の狂気カードが捨て山に加わります。
  • 追加枚数は出目に依存：100→6枚（特別演出） ／ 99〜95→5枚 ／ 94〜80→4枚 ／ 79〜50→3枚 ／ 49以下→2枚
  • 正気度が0になると「永久的狂気」となります。人間プレイヤーはゲームオーバー、AIは手番から脱落します。
  • 永久的狂気になったプレイヤーの勝利点は0となり、狂気カード1枚につき−1点が加算されます。
  • 正気度と汚染率はゲーム画面と戦闘準備画面に常時表示されます。

おすすめモード

設定でオンにするとAIが推奨する行動ボタンに「おすすめ」の吹き出しが表示されます。

  • 推奨基準：戦闘力9以上（追加ドローなし）または戦闘力が十分ある場合に神殿をおすすめ。
  • 追加ドロー+1は戦闘力+1として計算します。
  • 金貨1以下の場合は港をおすすめしません。
  • 警告が出る選択肢（戦闘力5以下・追加ドローなし、または金貨0）はおすすめになりません。

海底神殿に入る

  1. 手札を全て公開します。各探索者への装備は自動で割り当てられます。
    各探索者に武器1・防具1・道具1が装備されます（アイテムの重さ ≤ 探索者の筋力のものに限る）。
    火器（拳銃・自動小銃）は「火器使用可能」の探索者のみ装備されます。
  2. 神殿に積まれたカードをオープンします：
    • 宝物（宝冠・宝盃・宝剣）→ 即座に獲得、捨て山へ。
    • 秘宝（銀の鍵・輝くトラペゾヘドロン）→ 即座に獲得、ゲーム終了。
    • 神話生物・邪神の化身→ 戦闘へ。
  3. 戦闘前に追加ドローできます：
    手札の「ドロー枚数」の合計分、山札からカードを引けます（任意）。
    例）ハンドキャリーを持つと神殿で2枚追加ドローできます。引いた後、装備を自動再割り当てします。
  4. 戦闘を解決します：

    ①合計戦闘力を計算（全探索者の戦闘力 + 装備の戦闘力）

    ②狂気ペナルティ：手札の狂気カード枚数に応じた 2の累乗 のペナルティ
      （狂気1枚→2 / 狂気2枚→4 / 狂気3枚→8 / 狂気4枚→16…）
      旧神の印などの守護1毎に狂気1枚分を相殺（ペナルティを軽減）

    ③戦闘力 = 合計戦闘力 − 狂気ペナルティ

    ④戦闘力 ≥ 神話生物強さ → 勝利
      勝利：神話生物カードを捨て山へ ＋ 狂気1枚を捨て山へ

    ⑤戦闘力 ＜ 神話生物強さ → 敗北
      神話生物デッキのカード：デッキの一番下に戻り次のカードが現れる ＋ 狂気2枚
      秘宝デッキのカード：倒すまで居座り続ける ＋ 狂気2枚

  5. 邪神の化身と戦う場合：
    • 装備している宝物カード（宝冠・宝盃・宝剣）の相手が邪神の化身の場合、ボーナスが戦闘力に加算されます。
    • 遭遇時に狂気カード3枚が追加された後、邪神の化身がランダムで特殊行動を1つ実行します：
      武器破壊／防具破壊／狂気発動（+1枚）／狂気発動（+2枚）／静観／自己強化１（強さ+4）／自己強化２（強さ+8）／連続行動（低確率）／３連続行動（低確率）
      連続行動：6種の行動（武器破壊・防具破壊・狂気発動×2種・自己強化×2種）からランダムに2つを続けて実行します。
      ３連続行動：同6種の行動からランダムに3つを続けて実行します。
      行動後に戦闘力を再計算し、その結果で勝敗を判定します。
  6. 全手札を捨て山に置き、新たに8枚ドローして手番終了。

神殿の2種類のデッキ

神話生物デッキ：通常神話生物・宝物・邪神の化身1体が混在。倒せなかった場合はデッキ底に戻り、次のカードが現れます。

秘宝デッキ：残りの邪神の化身・秘宝が入っています。倒せなかった邪神の化身は居座り続け、倒すまで秘宝にたどり着けません。

港を訪れる

  1. 手札を全て公開します。
  2. 公開したカードの港効果が自動で発動し、金貨値を合計します。
    例）商人は「購入コスト-2・追加1枚購入」、補給船到着イベントは「追加1枚購入」が自動適用されます。
  3. 合計金貨値以下のコストのカードを購入できます。
    通常は1枚（商人・補給船到着等で増加）。
    購入先：基本カード・探索者カード・港カード（各山の一番上）。
    購入したカードは捨て山へ置かれます。未使用金貨は消滅します。
  4. 手札に精神科医探索者がいる場合、手札の狂気カードを全て破棄できます。
  5. 全てのカードを捨て山に置き、新たに8枚ドローして手番終了。

休息する

  1. 破棄（無料・最大2枚）：
    手札のカードをクリックして選択し（最大2枚）「選択した手札を破棄」ボタンで除外（デッキから永久に消える）。
    狂気カードのみ2枚まで、または通常カードのみ2枚までが可能（混合不可）。
    ※「安息の夜」イベント中は最大3枚まで破棄できます。
  2. 訓練（手札の探索者カードに訓練ボタンが表示されます）：
    探索者カードに直接表示された訓練ボタンをクリックすることで実行。
    手札の金貨値を合計し、💵6消費するごとに探索者の攻撃力を+1できます。
    初回の訓練のみ、攻撃力+1に加えて筋力も+1されます。2回目以降は攻撃力のみ上昇。
    探索者1人につき1回のみ。残余金貨がある限り別の探索者に続けて訓練可能。
    傭兵が手札にいる場合：訓練1回無料・攻撃力+2に変化。
  3. 精神科医（手札にいる場合）：
    手札に精神科医探索者がいる場合、「狂気カードを除去する」ボタンが表示されます。
    クリックで手札の狂気カードを全て除外できます（無制限）。
  4. 回復の泉（イベント中のみ）：
    「💧 回復の泉で水浴びをする」ボタンをクリックで手札の狂気カードを全て無料で除外できます。
  5. 全手札を捨て山に置き、新たに8枚ドローして手番終了。
