遊び方 — 海底神殿の秘宝

ゲームの目的

突如海上に浮上した海底神殿から秘宝を持ち帰ることが目的です。
秘宝（銀の鍵または輝くトラペゾヘドロン）を獲得した瞬間にゲームが終了し、
最終的に勝利点が最も多いプレイヤーが勝利します。

プレイヤー人数

2〜5人でプレイします。AIはノーマル・ハード・ストロングの3段階。

準備

各プレイヤーはスターターデッキ（12枚）を持ちます：

  • 初級探索者 ×6（攻撃1）
  • 古びた旧神の印 ×2（守護1・金貨値2）
  • ナイフ ×2（攻撃+1・金貨値1）
  • 初級防具 ×2（防御+1・金貨値2）

開始時に山札から8枚を手札として引きます。

先手番補正（第1ラウンドのみ）：後手ほど多く引き、手番差を補正します。
2番手: +1枚（計9枚）／3番手: +1枚（計9枚）／4番手: +2枚（計10枚）／5番手: +2枚（計10枚）

手番の流れ

スタートプレイヤーから時計回りに手番を行います。手番では以下から1つを選んで実行します：

  • 海底神殿に入る — モンスターと戦い宝を手に入れる
  • 港を訪れる — カードを購入してデッキを強化する
  • 休息する — カードを破棄・訓練してデッキを整える

ゲーム終了と勝利

いずれかのプレイヤーが秘宝（銀の鍵または輝くトラペゾヘドロン）を獲得した瞬間にゲームが終了します。
その後、全プレイヤーの勝利点を集計し、最多のプレイヤーが勝利します。
※輝くトラペゾヘドロンは「狂気カード1枚につき+1点」の特殊効果があります（獲得者の手札の狂気を集計）。

同点時のタイブレーク（順に適用）

  1. 秘宝を持っているプレイヤーが勝利
  2. 秘宝なしで同点の場合、宝剣・宝冠・宝盃を多く持つプレイヤーが勝利
  3. 上記も同数の場合、下位に秘宝を持つプレイヤーがいればその秘宝獲得者が勝利

ラウンドイベント

各ラウンドの開始時にラウンドイベントが1枚公開されます。イベントの効果はそのラウンド中の全プレイヤーに適用されます。
13枚のイベントカードをシャッフルして使用。尽きたら再シャッフルします。

  • 👹 モンスターの活性化（神殿系）：このラウンドの恐怖+2
  • ✨ 探索者の奮起（神殿系）：このラウンドの恐怖-2
  • 🔗 連続探索（神殿系）：神殿フェーズで勝利後、続けてもう1枚挑戦できる（2枚目は恐怖+2）
  • ⚓ 凪の港（港系）：このラウンドの購入コスト-1（下限1）
  • 🚢 補給船到着（港系）：このラウンドの購入回数+1
  • ⛈ 嵐の海峡（港系）：このラウンドの購入コスト+1
  • 🌙 安息の夜（休息系）：このラウンドの休息で破棄できるカードが最大3枚に
  • 💧 回復の泉（休息系）：このラウンドの休息で手札の狂気カードを全て除外できる（無料）
  • 📜 古の知識（休息系）：このラウンドの訓練コストが5Gになる
  • ⚡ 特訓の時（休息系）：このラウンドの訓練で攻撃力+2（コスト6G）
  • 🌊 幸運の潮流（全体系）：このラウンドはターン終了時のドロー枚数が9枚
  • 🔮 霊的な加護（全体系）：このラウンドの守護を持つ道具の守護値+1
  • 🌀 嵐のドロー（全体系）：このラウンドは手番開始時に手札を全て引き直す
  • 🌪 激動の日（全体系）：恐怖+2・神殿で追加ドロー1枚・守護+1が同時に発生する
  • ☀ 平穏の日（全体系）：このラウンドは特殊効果なし

デッキについて

山札が足りなくなったとき、捨て山をシャッフルして新しい山札にします。手番終了時は全手札を捨て山へ置き、新たに8枚ドローします。

海底神殿に入る

  1. 手札を全て公開します。各探索者への装備は自動で割り当てられます。
    各探索者に武器1・防具1・道具1が装備されます（アイテムの重さ ≤ 探索者の体力のものに限る）。
    火器（小火器・重火器）は「火器使用可能」の探索者のみ装備されます。
  2. 神殿に積まれたカードをオープンします：
    • 宝物（宝冠・宝盃・宝剣）→ 即座に獲得、捨て山へ。
    • 秘宝（銀の鍵・輝くトラペゾヘドロン）→ 即座に獲得、ゲーム終了。
    • モンスター→ 戦闘へ。
  3. 戦闘前に追加ドローできます：
    手札の「ドロー枚数」の合計分、山札からカードを引けます（任意）。
    例）ハンドキャリーを持つと神殿で2枚追加ドローできます。引いた後、装備を自動再割り当てします。
  4. 戦闘を解決します：

    ①合計戦闘力を計算（全探索者の攻撃 + 装備の攻撃 + 装備の防御）

    ②恐怖ペナルティ：恐怖基礎 = min(階層 × 2, 6)
      （階層1→2 / 階層2→4 / 階層3以上→6）
      旧神の印などの守護1毎にペナルティを-2ずつ軽減
      （守護1→-2軽減 / 守護2→-4軽減 / 守護3以上→ペナルティなし）

    ③実質戦闘力 = 合計戦闘力 − 恐怖ペナルティ

    ④実質戦闘力 ≥ モンスター強さ → 勝利
      勝利：モンスターカードを捨て山へ ＋ 狂気1枚を捨て山へ

    ⑤実質戦闘力 ＜ モンスター強さ → 敗北
      モンスターデッキのカード：デッキの一番下に戻り次のカードが現れる ＋ 狂気2枚
      秘宝デッキのカード：倒すまで居座り続ける ＋ 狂気2枚

  5. 強さ25の強敵（ボス）と戦う場合、装備している宝物カード（宝冠・宝盃・宝剣）の対ボスボーナスが戦闘力に加算されます。
  6. 全手札を捨て山に置き、新たに8枚ドローして手番終了。

神殿の2種類のデッキ

モンスターデッキ：通常モンスター・宝物・ボス1体が混在。倒せなかった場合はデッキ底に戻り、次のカードが現れます。

秘宝デッキ：残りのボスモンスター・秘宝が入っています。倒せなかったボスは居座り続け、倒すまで秘宝にたどり着けません。

港を訪れる

  1. 手札を全て公開します。
  2. 公開したカードの港効果が自動で発動し、金貨値を合計します。
    例）バックパックは「港でカードを2枚購入できる」効果が自動適用されます。
  3. 合計金貨値以下のコストのカードを購入できます。
    通常は1枚（バックパック等で増加）。
    購入先：基本カード・探索者カード・港カード（各山の一番上）。
    購入したカードは捨て山へ置かれます。未使用金貨は消滅します。
  4. 手札に精神科医探索者がいる場合、手札の狂気カードを全て破棄できます。
  5. 全てのカードを捨て山に置き、新たに8枚ドローして手番終了。

休息する

  1. 精神科医（手札にいる場合）：
    手札に精神科医探索者がいる場合、手札の狂気カードを全て除外できます（無制限）。
    ※港フェーズでも同様に使用できます。
  2. 破棄（無料・最大2枚）：
    手札から2枚まで除外（デッキから永久に消える）。
    狂気カードのみ2枚まで、または通常カードのみ2枚までが可能（混合不可）。
  3. 訓練（金貨値6消費・探索者の攻撃力+1）：
    手札の金貨値を合計し、6G消費するごとに探索者の攻撃力を+1できます。
    初回の訓練のみ、攻撃力+1に加えて体力も+1されます。2回目以降は攻撃力のみ上昇。
    残余金貨がある限り続けて訓練可能。
  4. 全手札を捨て山に置き、新たに8枚ドローして手番終了。

宝物カード（モンスターデッキに混在）

宝物は道具スロットに装備でき、強さ25のボスと戦うとき対ボスボーナスが戦闘力に加算されます。

  • 宝冠：勝利点4 / 金貨値7 / 対ボス戦闘力+8
  • 宝盃：勝利点4 / 金貨値6 / 対ボス戦闘力+10
  • 宝剣：勝利点4 / 金貨値5 / 対ボス戦闘力+12

秘宝カード（秘宝デッキ）

秘宝デッキにはボスモンスターも混在しており、秘宝に到達するにはボスを倒す必要があります。

  • 銀の鍵：勝利点10 — 獲得即ゲーム終了
  • 輝くトラペゾヘドロン：勝利点7 ＋ 狂気カード2枚につき+1点 — 獲得即ゲーム終了
